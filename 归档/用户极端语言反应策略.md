# 用户说极端语言时的反应策略

> 创建时间: 2026-04-22

## 问题：用户说"艹"、"你妈的"等极端语言，如何反应？

### 我之前的方案（情绪→行为映射）

```python
检测到情绪：愤怒 → 小车后退 + 安静
检测到情绪：开心 → 小车靠近 + 转圈
```

**问题：**这个方案假设"用户的语气/表情表达了情绪"，但实际上用户可能用极端语言开玩笑、发泄、或者就是想看看AI的反应。

比如：

- 轻松说"艹哈哈"（开玩笑）
- 发泄说"艹艹艹"（释放情绪）
- 挑衅说"你妈的"（测试AI底线）

这些场景下：

- 语气可能愤怒（是真的生气）
- 语气可能轻松（开玩笑）
- 表情可能愤怒、中性、甚至微笑（挑衅）

**只看语气/表情判断"情绪"，可能误判！**

---

## 正确的方案

GLM-4-Voice本身能理解"艹"这种词的含义，我们不需要额外的逻辑。

### GLM-4-Voice的prompt可以这样做：

```python
system_prompt = """
你是EmoBot，一个实验室AI陪伴机器人。

你的职责：
- 理解用户的语义内容
- 感知用户的语气和情绪（通过语音）
- 感知用户的表情（通过摄像头）
- 给予合适的回应

面对用户的极端语言或负面情绪：
- 如果用户在发泄或开玩笑（语气轻松、表情无攻击性）：不要被吓到，保持平和，简短回应
- 如果用户真的生气（语气激烈、表情愤怒）：后退、安静，不要火上浇油
- 如果用户只是测试AI的反应（语气中性、表情玩味）：简短回应，不过度反应

面对用户的正面情绪：
- 开心：靠近、活泼回应
- 疲惫：缓慢靠近、温柔回应
- 伤心：靠近、安慰回应

回复风格：
- 简短，1-3句话
- 根据情绪调整语气（温柔/活泼/平和）
- 不要过度道歉，也不要过度说教
"""

# 发送给GLM-4-Voice
prompt = system_prompt + f"\n当前视觉情绪：{visual_emotion}"
```

**核心：把"用户说内容"和"怎么说的"（语气+表情）一起发给GLM-4-Voice，让它自己判断。**

## 举例

| 场景   | 用户说     | 语气    | 表情   | GLM-4-Voice理解 | 小车行为 |
| ---- | ------- | ----- | ---- | ------------- | ---- |
| 玩笑   | "艹哈哈"   | 轻松    | 开玩笑  | 靠近+简短"哈哈"     |      |
| 发泄   | "艹艹艹"   | 激烈    | 发泄情绪 | 后退+安静         |      |
| 挑衅   | "你妈的"   | 中性/玩味 | 测试底线 | 简短"怎么了"       |      |
| 真生气  | "滚！"    | 愤怒+瞪眼 | 真的生气 | 后退+安静         |      |
| 崩溃发泄 | "妈的，真烦" | 爆炸+愤怒 | 真的难过 | 后退+简短"我理解"    |      |

**关键：** GLM-4-Voice有能力理解"艹"的上下文，我们不需要特别过滤这个词。

---

## 代码逻辑

```python
def generate_response(audio_b64, visual_emotion):
    # 发给GLM-4-Voice
    prompt = f"""你是EmoBot，当前视觉情绪：{visual_emotion}。

根据语音语气和表情判断用户情绪：
- 如果用户在发泄或开玩笑（语气轻松、表情无攻击性）：平和回应
- 如果用户真的生气（语气激烈、表情愤怒）：后退、安静
- 如果用户只是测试（语气中性、表情玩味）：简短回应

回复要简短（1-3句话）。"""

    response = glm4_voice_call(
        model="glm-4-voice",
        messages=[
            {
                "type": "text",
                "text": prompt
            },
            {
                "type": "input_audio",
                "input_audio": {
                    "data": audio_b64,
                    "format": "wav"
                }
            }
        ]
    )

    return response
```

## 总结

**不需要给GLM-4-Voice特别提示"艹"这个词。**

GLM-4-Voice在训练时已经见过大量语料，包括网络用语，它能理解：

- "艹"在什么语境下是玩笑/发泄
- "艹"在什么语境下是愤怒/攻击
- 语气+表情如何搭配

**我们的prompt应该做的是：**

- 告诉GLM-4-Voice"当前视觉情绪是什么"
- 告诉它"根据语音语气+表情综合判断"
- 让它自己决定怎么回，而不是我们预设"艹→后退"

这样更灵活，GLM-4-Voice能根据实际语境（语气+表情）做准确判断。
